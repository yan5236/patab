PaTab 离线安装教程

如果你暂时无法通过 Chrome 应用商店或 Edge 外接程序商店安装 PaTab，可以按照以下步骤手动加载离线安装包。整个过程只需 1 分钟左右。

适用浏览器：Google Chrome、Microsoft Edge 及其他基于 Chromium 内核的浏览器（如 360 极速浏览器、QQ 浏览器等）。

第一步：进入扩展管理页面

点击浏览器右上角的 ⋮（三个点） 菜单，依次选择 「扩展程序」→「管理扩展程序」。

第二步：开启开发者模式

在扩展管理页面右上角，找到 「开发者模式」 开关。如果当前是关闭状态，点击将其启用。

第三步：加载离线安装包

将下载好的 .zip 文件直接拖入扩展管理页面，浏览器会自动识别并弹出安装提示。

备选方式：也可以先将 zip 解压到任意文件夹，然后点击页面左上角的 「加载已解压的扩展程序」，选中解压后的文件夹即可。

第四步：确认安装

在弹出的确认窗口中点击 「添加扩展程序」。稍等片刻，当扩展列表中出现了 PaTab 的卡片信息，说明安装已成功。

接下来打开一个新标签页验证效果：

  • 如果标签页自动替换为 PaTab 的界面——搞定！
  • 如果浏览器提示「标签页已被其他扩展程序修改」，点击 「保留现状」 即可保留 PaTab 的更改。

更新扩展

如果你已经通过离线安装包安装了 PaTab，后续要更新到新版本时，只需下载新的离线安装包，然后按照上述步骤重新安装一遍即可覆盖旧版本。

建议定期访问下载页面获取最新的离线安装包，以获得功能更新和问题修复。

常见问题

Q：拖入 zip 后没有反应？

请确认开发者模式已开启（第二步）。如果拖入方式始终不生效，可以试试备选方案：将 zip 解压后，点击「加载已解压的扩展程序」并选中解压文件夹。

Q：安装后新标签页没变化？

检查扩展列表中 PaTab 是否已启用（开关为蓝色）。如果已启用但仍未生效，尝试重启浏览器。

Q：Edge 浏览器操作一样吗？

完全一样。Edge 的扩展管理页面入口为：「⋮ 菜单」→「扩展」→「管理扩展」，其余步骤与 Chrome 一致。
